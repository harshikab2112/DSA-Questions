# Advanced Tree Data Structures Coding Questions

This repository contains a collection of **coding questions** on **Advanced Tree Data Structures**, ranging from **beginner to advanced** levels. These problems will help you strengthen your problem-solving skills in tree-based data structures.

### **Binary Search Tree (BST)**

---

1. Implement insertion in a Binary Search Tree (BST).
2. Implement deletion in a BST.
3. Implement search operation in a BST.
4. Find the minimum and maximum value in a BST.
5. Find the height of a BST.
6. Find the in-order predecessor and successor of a node in BST.
7. Check if a given tree is a BST.
8. Convert a sorted array to a balanced BST.
9. Convert a BST to a sorted doubly linked list.
10. Find the Lowest Common Ancestor (LCA) in a BST.

### **AVL Tree**

---

11. Implement insertion in an AVL tree.
12. Implement deletion in an AVL tree.
13. Implement search operation in an AVL tree.
14. Find the height of an AVL tree.
15. Perform left and right rotations in an AVL tree.
